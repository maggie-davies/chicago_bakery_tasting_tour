# Assignment3_chicago_bakery_trail

For this assignment, I researched and mapped acclaimed Chicago bakeries to create a tasting trail to recommend to my brother, who will be entering a baking competition there later this year. While it is an amateur baking competition, the stakes are rather high, as he has challenged himself with creating dishes in five separate baked goods categories. I created this list to help him gather ideas and enjoy tasting what Chicago has to offer along the way!

While I only included my top ten locations on this map, I have a longer list of 22 locations that I may build out in the future. I'd also like to include more details, images, and interactivity in my pop-ups, including a website link and recommended dishes to try.
